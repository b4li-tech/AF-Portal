# OPERATOR FRAMEWORK

## Definicja

Operator Framework jest podmiotem wykorzystującym Akcelerator Framework do prowadzenia analiz, syntezy oraz walidacji.

Operatorem może być człowiek, sztuczna inteligencja lub zespół współpracujących operatorów.

## Zadania Operatora

* definiowanie Obiektu Analizy
* prowadzenie analiz
* tworzenie syntez
* przeprowadzanie walidacji
* rozwijanie Biblioteki Wiedzy

## Cel

Budowa coraz lepszych modeli rzeczywistości poprzez wykorzystanie narzędzi Frameworka.
