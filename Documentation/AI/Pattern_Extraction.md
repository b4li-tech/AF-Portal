# PATTERN EXTRACTION

## Definicja

Pattern Extraction jest procesem wykrywania powtarzalnych wzorców występujących w analizach.

## Cel

Identyfikacja elementów pojawiających się wielokrotnie w różnych Obiektach Analizy.

## Wynik

Rezultatem procesu jest Wzorzec, który może zostać zapisany w Bibliotece Wiedzy Frameworka.

## Przykłady

* powtarzalne zachowania
* powtarzalne procesy
* powtarzalne mechanizmy działania
* powtarzalne schematy rozwoju
