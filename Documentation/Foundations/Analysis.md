# ANALYSIS

## Definicja

Analiza jest procesem badania Obiektu Analizy w celu lepszego zrozumienia jego budowy, działania, oddziaływania oraz potencjalnych kierunków rozwoju.

## Cel

Zmniejszenie niepewności poprzez pozyskanie informacji oraz budowę modelu badanego obiektu.

## Wynik

Rezultatem analizy jest zbiór obserwacji, wniosków oraz hipotez poddawanych dalszej syntezie i walidacji.

## Powiązania

* Observation
* Tor A
* Tor B
* Tor C
* Synthesis
