# PATTERN

## Definicja

Wzorzec (Pattern) jest powtarzalną strukturą, zależnością, zachowaniem lub mechanizmem występującym w wielu analizach.

## Cel

Identyfikacja elementów powtarzających się niezależnie od Obiektu Analizy.

## Założenie

Im częściej dany element pojawia się w różnych analizach, tym większe prawdopodobieństwo, że stanowi użyteczny wzorzec.

## Źródła Wzorców

* analizy
* obserwacje
* syntezy
* korelacje
* walidacje

## Wynik

Wzorzec może zostać zapisany w Bibliotece Wiedzy i wykorzystywany podczas kolejnych analiz.

## Relacje

Wzorzec jest naturalnym przeciwieństwem Anomalii.

Anomalia może prowadzić do powstania nowego Wzorca.

Wzorzec może pomagać w wykrywaniu nowych Anomalii.
