# INTERFACES

## Definicja

Interfejsy AF są sposobami obserwacji, interpretacji oraz komunikacji pomiędzy operatorem a Obiektem Analizy.

Interfejs nie zmienia Obiektu Analizy.

Interfejs wpływa na sposób jego postrzegania.

---

# Cel

Zwiększenie liczby perspektyw dostępnych podczas analizy.

---

# Funkcja

Interfejsy pozwalają:

* rozszerzać obserwację
* ograniczać błędy interpretacyjne
* ujawniać ukryte zależności
* porównywać różne modele rzeczywistości
