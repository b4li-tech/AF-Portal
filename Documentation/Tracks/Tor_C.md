# TOR C

## Cel

Tor C odpowiada za analizę potencjalnych kierunków rozwoju, konsekwencji oraz możliwości wynikających z Obiektu Analizy.

Tor C jest torem eksploracji przyszłości.

## Pytanie Główne

Co dalej z tego wynika?

## Obszary Analizy

* możliwości
* potencjał
* rozwój
* konsekwencje
* zagrożenia
* szanse
* kierunki zmian
* przyszłe zastosowania

## Wynik

Wynikiem Toru C jest zbiór potencjalnych scenariuszy, kierunków rozwoju oraz dalszych pytań badawczych.

## Przykład

Obiekt Analizy: Suno AI

Tor C analizuje:

* wpływ na przemysł muzyczny
* możliwość odtwarzania utworów na żywo
* nowe zawody związane z AI
* rozwój własnych linii produkcji muzycznej
* nowe modele współpracy człowiek–AI

Tor C nie opisuje stanu obecnego.

Tor C bada możliwe przyszłości wynikające z aktualnego stanu.
