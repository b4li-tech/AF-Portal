# VALIDATION MODULE

## Definicja

Moduł Walidacji odpowiada za ocenę jakości wyników uzyskanych podczas pracy Frameworka.

---

# Cel

Ograniczenie błędów oraz zwiększenie wiarygodności wiedzy.

---

# Kryteria

* spójność logiczna
* zgodność z obserwacjami
* powtarzalność
* użyteczność
* odporność na krytykę

---

# Wynik

Ocena jakości modelu oraz decyzja o jego przyjęciu lub odrzuceniu.

---

# Relacje

Walidacja następuje po Syntezie i poprzedza zapis do Biblioteki Wiedzy.
